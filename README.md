# KDialog

[![MIT License](https://img.shields.io/badge/License-MIT-a10b31)](LICENSE)

**KDialog** is a work-in-progress KDialog wrapper for Go.

This package, to my knowledge, is the most complete KDialog wrapper for Go, consisting of full support for every dialog box type and response.

![image](https://github.com/user-attachments/assets/fc03c7eb-ec33-434a-b992-a3e89584c842)

> [!WARNING]
> This package is very much unfinished, this is an incomplete package that has bugs and code that needs to be refined. Contribute to the package [here](https://github.com/notwithering/kdialog/pulls) if you would like.
